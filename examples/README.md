# UI/UX 优化示例

此文件夹包含 UI/UX 优化的参考示例，**不影响项目运行**。

## 📁 文件夹结构

```
examples/
├── refactored-pages/        # 页面重构示例
│   ├── CloudDesktop.example.tsx
│   ├── OperationsManagement.example.tsx
│   └── README.md
└── component-usage/         # 组件使用示例
    └── FormExample.tsx
```

## 🎯 用途

这些示例文件展示了如何应用统一的设计规范：

1. **重构示例** - 展示如何将现有页面迁移到新的设计标准
2. **组件用法** - 演示标准组件的使用方式
3. **最佳实践** - 提供代码参考和实现思路

## 🚀 如何使用

1. 阅读 [UI_STANDARDS.md](../docs/UI_STANDARDS.md) 了解设计规范
2. 查看示例代码理解实现方式
3. 将设计模式应用到实际项目中
4. 根据需求调整和优化
