# 设计系统参考组件

⚠️ **此文件夹仅供参考，不要在项目中直接导入使用**

## 📁 文件说明

这个文件夹包含统一设计规范的参考实现，用于指导项目的 UI/UX 优化。

### 核心组件

- **PageLayout.tsx** - 统一的页面布局组件
- **TableToolbar.tsx** - 标准化的表格工具栏
- **StandardTable.tsx** - 标准化的数据表格
- **DataCount.tsx** - 统一的数据统计文本
- **StandardForm.tsx** - 标准化的表单布局

## 🎯 使用方式

这些组件是**参考实现**，展示了如何应用统一的设计规范。你可以：

1. **参考代码结构** - 了解如何使用 tokens 替代硬编码
2. **复制代码片段** - 将需要的部分复制到实际组件中
3. **学习最佳实践** - 理解组件抽象和复用的思路
